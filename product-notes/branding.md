# Branding

This document captures the initial visual identity direction for scaudit.

## Logo Direction

The proposed logo combines:

- A magnifying glass, representing audit, inspection, review, and traceability.
- A single-cell scatter / embedding pattern, representing scRNA-seq data and cellular heterogeneity.
- Evidence/report icons, representing structured annotation evidence, validation, and decision records.
- The wordmark `scaudit`.
- The subtitle `SINGLE CELL AUDIT`.
- The tagline:

```text
EVIDENCE • EXPLAINED • AUDITABLE
```

## Visual Meaning

The logo aligns well with the project identity:

- **Single-cell data**: represented by the colored cell-like dot cloud.
- **Audit and review**: represented by the magnifying glass.
- **Evidence-based decisions**: represented by the checklist/report elements.
- **Trust and reproducibility**: represented by the structured visual layout.

## Color Direction

The current logo uses:

- Deep navy for wordmark, outlines, and technical structure.
- Purple, blue, teal, and green accents for biological diversity and multi-source evidence.
- White background for clarity and scientific presentation.

These colors can become the base palette for:

- CLI/report visual identity.
- Quarto report styling.
- Plotly figure defaults.
- GitHub README assets.
- Future web UI.

## Usage Notes

The logo is intended to be used as the primary scaudit logo for:

- GitHub README.
- Documentation.
- Quarto reports.
- Presentation slides.
- Future web app or landing page.
